Документация Протоколов

Архитектура системы

Интерфейс построен на базе Arctic Node OS. Все данные передаются по зашифрованному каналу L-256. Взаимодействие с датчиками происходит по протоколу MQTT.

Безопасность

В случае падения температуры в жилом модуле ниже +15°C, система автоматически активирует форсированный режим обогрева.

API Доступ

Для внешних разработчиков доступен эндпоинт /v1/telemetry для получения логов в формате JSON.
